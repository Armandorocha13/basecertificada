# Sprint 01 - Setup e Interface do Sistema de Auditoria Logística

**Contexto:**
Iniciando um novo projeto. Esta pasta de `prompts` armazenará todo o histórico de sprints para que o contexto nunca seja perdido.

**Requisitos da Sprint e Design:**
- Atuação como Frontend Sênior.
- Interface web moderna, limpa e responsiva (Sistema de Auditoria Logística).
- Stack: HTML5 + Tailwind CSS + Vanilla JS (usando Vite para a build e servidor local).
- Identidade visual estritamente profissional e corporativa: Branco com detalhes em preto. Cores neutras, bom contraste, acessibilidade e simplicidade com foco total na funcionalidade.

**Requisitos da Tela (Formulário):**
- Formulário centralizado.
- Input de texto: 'Nome da Base'.
- Input de seleção: 'Nome do Auditor'.
- 5 Campos de Upload de Arquivo com labels claras:
  1. Vídeo do Estoque (`accept="video/*"`)
  2. PDF Patrimônio Aniel (`accept=".pdf"`)
  3. PDF Documentos de Liberação (`accept=".pdf"`)
  4. Excel Relatório de Seriais (`accept=".xlsx, .xls"`)
  5. Excel Conferência de Notas Fiscais (`accept=".xlsx, .xls"`)
- Botão de envio 'Enviar Auditoria', o qual deve iniciar bloqueado/desabilitado.
